**🔑 Key Concepts (Answers)**

**C++ Basics**

A program = instructions the computer runs
Pressing run = code is compiled → program executes → starts at main()
main() = entry point of the program
std::cout = sends output to the console
No main() = no entry point → program will not build/run
Git & GitHub Basics
Git = tool that tracks changes locally
GitHub = platform to store code online
Repo = project folder with version history
git init = initializes Git in a folder
git add . = stages files for commit
git commit = saves a snapshot of changes
git remote add origin = links local repo to GitHub
git push = uploads commits to GitHub
Repo link = URL of your GitHub repository
