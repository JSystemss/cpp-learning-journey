Loops (Key Concepts)
Loop = runs code repeatedly while condition is true
for loop = used when number of iterations is known
while loop = used when condition controls repetition
Infinite loop = condition never becomes false
% (modulo) = gives remainder after division
Used for checking even/odd numbers

What is an index?

Why do we use loops with arrays?
What happens if you access an index out of bounds?

Arrays (Answers)

Array = a variable that stores multiple values
Index = position of a value in the array
Indexing starts at 0
Loops allow access to all elements efficiently
Accessing out of bounds = undefined behavior (can crash or give garbage value)

Arrays (Key Insight)

Arrays store values next to each other in memory (contiguous memory)
Each element is accessed using an index
C++ does NOT protect against invalid index access
